hshort
======
A canonical URL shortener written using Haskell and Scotty. Nothing to see here.

[![screenshot](/screenshot.png)](http://hshort.co)

## Docker and tutum.co
A `tutum.yaml` file is provided in this repository, so you can `tutum up` this
service.
